# 关于开展计算机学院2026届本科毕业论文（设计）中期检查工作的通知

日期：2026-01-21 来源： 作者： 浏览：1547 次

各位老师及同学：

为了进一步加强毕业论文（设计）各环节的管理，提高毕业论文（设计）质量，根据《四川大学本科毕业论文（设计）管理办法（修订）》（川大教[2022]56号文件）及工作程序的安排，计算机学院将组织开展毕业论文（设计）中期检查工作，现将检查内容及有关注意事项通知如下：

1、请指导教师根据学生“毕业论文（设计）任务书”的安排，对各项任务的完成情况进行认真检查。

2、请学生在2026年3月 16 日晚上12：00前，将**文献综述，外文文献翻译，中期报告（附件1）**提交至本科毕业论文（设计）管理系统。（**提示：根据学校文件要求，外文文献翻译100%完成，实验进展和文献综述至少完成50%**）。

3、请指导教师在3月23日中午12：00前登录本科毕业论文（设计）管理系统，完成学生文献综述、外文文献翻译与中期报告的审核工作。

4、请指导教师按照《四川大学本科毕业论文（设计）管理办法（修订）》（川大教[2022]56号文件）文件要求（附件2），检查学生论文格式及内容，督促毕业论文进度。

5、学院组织相关专家，就毕业论文（设计）的文献综述、外文文献翻译、实验进展以及指导教师工作情况等方面进行认真检查，并完成毕业论文（设计）中期检查工作总结。

6、请指导教师填写“2026届毕业论文（设计）中期检查情况汇总表”（附件3），于3月27日17:00前以电子文档形式发送到wyl304@scu.edu.cn邮箱报教学科备案。

附件：1、四川大学毕业论文（设计）中期检查表

2、四川大学本科毕业论文（设计）管理办法（修订）-川大教【2022】56号

3、2026届毕业论文（设计）中期检查情况汇总表
