# 毕设材料目录

本目录集中存放本科毕业设计中期检查相关材料。

## 推荐阅读顺序

1. [`中期材料提交.md`](./中期材料提交.md)
   - 学院通知与系统表单要求
2. [`中期检查提交内容汇总.md`](./中期检查提交内容汇总.md)
   - 可直接用于系统填写的中期检查与文献综述内容
3. [`中期材料调研提示词.md`](./中期材料调研提示词.md)
   - 给 `Manus` 等智能体使用的调研提示词
4. [`manus生成内容/主题驱动的多源视频自动编排系统设计与实现文献综述.md`](./manus生成内容/主题驱动的多源视频自动编排系统设计与实现文献综述.md)
   - 已手工清洗后的文献综述版本
5. [`manus生成内容/外文文献论文基本信息及翻译建议.md`](./manus生成内容/外文文献论文基本信息及翻译建议.md)
   - 外文翻译选文与章节建议

## 当前建议

1. 系统填写时，优先使用 [`中期检查提交内容汇总.md`](./中期检查提交内容汇总.md)。
2. 文献综述正文以 `manus生成内容` 中已清洗版本为准。
3. 外文翻译材料已单独准备，不与本目录中的中期检查正文混写。
