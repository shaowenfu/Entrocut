# “主题驱动的多源视频自动编排系统”文献调研补充材料

## 1. 本科毕业设计中期材料文献综述提纲

### 第一章：引言
- 1.1 研究背景与意义：短视频时代的创作爆发与人工编排的效率瓶颈。
- 1.2 课题核心目标：多源视频素材的主题驱动式自动化编排。
- 1.3 核心技术挑战：语义一致性检索、素材自动切分、AI Agent 决策闭环、长上下文组装。

### 第二章：视频检索与摘要技术研究现状
- 2.1 文本-视频跨模态检索：从经典特征对齐到 CLIP 等大规模预训练模型的演进。
- 2.2 视频摘要与关键帧提取：基于时空注意力机制的内容提炼技术。
- 2.3 镜头边界检测：基于深度学习的突变与渐变转场识别。

### 第三章：自动化视频剪辑与编排逻辑
- 3.1 智能剪辑算法：基于语义标签的素材筛选与排序。
- 3.2 叙事一致性建模：如何保证多源素材在主题下的视觉与逻辑连贯。

### 第四章：大模型驱动的 AI Agent 架构与系统工程
- 4.1 LLM-based Autonomous Agents：感知、规划、行动（Action）的核心框架。
- 4.2 Agent 运行时与工具调用：闭环执行（Tool Calling Loop）与错误反馈机制。
- 4.3 系统管线工程：长上下文组装（Context Assembly）与服务端高并发能力设计。

### 第五章：总结与展望
- 5.1 当前技术的局限性：生成式 AI 对剪辑逻辑的可控性不足。
- 5.2 未来研究方向：端到端的 Agent 驱动视频生产流水线。

## 2. 各主题块之间的关系图（逻辑结构描述）

系统逻辑可视为一个**“意图驱动的闭环控制系统”**：

1.  **输入层 (User Intent)**：用户输入主题（Text Query）。
2.  **规划层 (Agent Planner)**：AI Agent 解析主题，拆解为检索任务、切分任务、编排任务。
3.  **执行层 (Tools/Modules)**：
    -   **检索模块**：利用 *Text-to-Video Retrieval* 从库中捞取原始素材。
    -   **处理模块**：利用 *SBD/Segmentation* 将素材切分为原子镜头。
    -   **精炼模块**：利用 *Video Summarization* 提取最符合主题的片段。
4.  **反馈层 (Agent Runtime)**：Agent 监控各工具输出，进行 *Context Assembly*，若结果不佳则触发重试或策略调整（闭环）。
5.  **输出层 (Orchestrated Video)**：最终生成符合叙事逻辑的视频。

## 3. “实验进展”与“其它情况”撰写建议

### 实验进展 (Experimental Progress)
-   **算法验证**：已初步跑通基于 CLIP 的视频检索 Demo，检索准确率（Recall@K）达到预期。
-   **Agent 运行时搭建**：完成了基于 FastAPI 的 Agent 服务端基础架构，实现了工具注册与动态调用的闭环逻辑。
-   **上下文组装策略**：实现了针对视频元数据的上下文动态组装方案，能够有效减少 LLM 调用的 Token 消耗。
-   **系统联调**：目前已完成“文本检索 -> 视频切分 -> Agent 编排逻辑”的初步全链路测试。

### 其它情况 (Other Issues/Challenges)
-   **性能瓶颈**：长视频的实时切分与特征提取计算开销较大，后续计划引入 GPU 加速与异步处理队列。
-   **指令遵循性**：LLM 在处理复杂剪辑指令时偶尔出现幻觉，需进一步优化 Prompt Engineering 或引入少样本学习（Few-shot）。
-   **工程细节**：服务端能力设计中，关于视频文件的流式传输与 Agent 状态保持仍需细化。

## 4. 如何将 AI Agent 纳入课题叙事（不偏题建议）

**叙事策略：将 AI Agent 定义为“系统的大脑”与“流程的调度器”。**

-   **核心论点**：传统的视频编排系统是“硬编码（Hard-coded）”的，无法处理多变的用户意图和海量的异构素材。
-   **Agent 的角色**：
    -   **解耦合**：Agent 将视频处理工具（检索、切分、摘要）与业务逻辑解耦，通过自然语言指令动态组合工具。
    -   **动态规划**：针对不同的“主题”，Agent 能够生成不同的编排路径（例如：新闻类主题侧重叙事，运动类主题侧重节奏）。
    -   **闭环纠错**：在自动编排过程中，Agent 能根据素材的质量反馈（如分辨率不足、语义不匹配）自动调整检索策略。
-   **术语衔接**：在撰写时，将“系统设计”描述为“基于 Agent 架构的智能编排引擎”，将“工具调用”描述为“编排原子操作的动态映射”。这样既保留了视频处理的核心业务，又突出了 AI Agent 作为先进生产力工具的技术亮点。
