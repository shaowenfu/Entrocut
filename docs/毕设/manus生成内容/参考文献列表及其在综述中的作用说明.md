## 参考文献在综述中的作用说明

1.  **[1] A Review of Video Text Retrieval Research (2026)**：作为 Text-to-Video Retrieval 部分的最新综述，提供了该领域的研究背景、分类和最新进展，为本课题的视频检索模块提供了宏观的理论支撑。
2.  **[2] CLIP4Clip: An Empirical Study of CLIP for End to End Video Clip Retrieval (2021)**：作为 Text-to-Video Retrieval 领域的经典基础论文，详细阐述了如何将 CLIP 模型应用于视频检索，为本课题的跨模态检索技术选型提供了核心方法论。
3.  **[3] VideoCLIP 2.0: An Interactive CLIP-Based Video Retrieval System (2024)**：在 Text-to-Video Retrieval 部分引用，说明了在实际系统中如何优化 CLIP-based 检索的效率和交互性，与本课题“多源”和“系统”的特点紧密相关。
4.  **[4] A Comprehensive Survey of Video Summarization Techniques with Emphasis on Real-World Applications (2025)**：作为 Video Summarization 部分的最新综述，全面介绍了视频摘要的各类方法及其在现实世界中的应用，为本课题的视频摘要模块提供了理论框架。
5.  **[5] Deep Learning for Video Summarization: Systematic Literature Review (2026)**：作为 Video Summarization 部分的系统文献综述，提供了深度学习在视频摘要中应用的详细分析和分类，有助于理解该领域的技术演进和趋势。
6.  **[6] Scene classification for sports video summarization using transfer learning (2020)**：在 Video Summarization 部分引用，作为早期但具有代表性的工作，说明了如何利用迁移学习进行特定场景的视频摘要，为本课题的素材提炼提供了实践参考。
7.  **[7] Video shot-boundary detection: issues, challenges and solutions (2024)**：作为 Shot Boundary Detection / Segmentation 部分的综述，总结了镜头边界检测的挑战和解决方案，为本课题的视频切分模块提供了全面的技术背景。
8.  **[8] Better Learning Shot Boundary Detection via Multi-task (2021)**：在 Shot Boundary Detection / Segmentation 部分引用，作为该领域的基础论文，强调了多任务学习在提升 SBD 性能中的作用，有助于理解 SBD 的技术细节。
9.  **[9] Shot Boundary Detection Using Color Clustering and CCSR (2023)**：在 Shot Boundary Detection / Segmentation 部分引用，介绍了具体的 SBD 算法，为本课题的视频切分技术选型提供了具体方法。
10. **[10] AI Video Editing: a Survey (2022)**：作为 Automatic Video Editing 部分的基础综述，梳理了 AI 在视频剪辑中的应用历史和自动化编排的逻辑框架，为本课题的自动剪辑模块提供了高层次的指导。
11. **[11] Design of an integrated model for video summarization using multimodal fusion and YOLO for crime scene analysis (2025)**：在 Automatic Video Editing 部分引用，展示了多模型集成（如 YOLO 和多模态融合）在复杂编排任务中的应用，为本课题的编排系统设计提供了实践案例。
12. **[12] AI Agent Systems: Architectures, Applications, and Evaluation (2026)**：作为 System / Pipeline Engineering 部分的基础论文，提供了工业级 AI Agent 系统的全生命周期设计和工程化实践，为本课题的系统架构设计提供了宏观指导。
13. **[13] A Survey on Large Language Model based Autonomous Agents (2024)**：作为 AI Agent Design and Development 部分的核心综述，详细阐述了 LLM 驱动的自主智能体架构、规划与执行能力，是本课题引入 AI Agent 的主要理论依据。
14. **[14] Agent Operating Systems (Agent-OS): A Blueprint Architecture (2025)**：在 AI Agent Design and Development 部分引用，探讨了 Agent 运行时的资源调度、安全隔离和工具调用闭环机制，与本课题的“Agent 运行时”和“工具调用闭环”需求直接相关。
15. **[15] The landscape of emerging AI agent architectures for reasoning and tool calling (2024)**：在 AI Agent Design and Development 部分引用，对比了不同 Agent 架构在推理规划与工具调用方面的优劣，为本课题的 Agent 决策逻辑提供了参考。
16. **[16] Dynamic Planning and Tool Use in Next-Gen AI Agents (2026)**：在 System / Pipeline Engineering 部分引用，强调了 Agent 在执行复杂任务时的动态规划调整和闭环执行能力，为本课题的服务端能力设计提供了重要思路。
