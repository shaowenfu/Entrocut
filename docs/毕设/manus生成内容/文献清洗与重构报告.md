## Part 3. 文献综述提纲（本科毕设中期版）

### 1. 引言
- 1.1 课题背景：短视频创作爆发与自动化编排需求。
- 1.2 系统目标：实现主题驱动、多源素材、智能编排的闭环系统。
- 1.3 核心架构演进：从传统硬编码剪辑向 AI Agent 驱动的智能化工作流转变。

### 2. 视频处理基础技术研究现状
- 2.1 文本-视频跨模态检索：CLIP 等视觉语言预训练模型在多源素材定位中的应用。
- 2.2 视频摘要与内容提炼：从视觉特征提取到 LLM 语义辅助的摘要生成。
- 2.3 镜头边界检测：作为编排原子单位的视频自动切分技术。

### 3. 自动化视频剪辑与编排逻辑
- 3.1 智能剪辑综述：自动化排序、节奏控制与叙事一致性。
- 3.2 主题驱动的语义编排：如何将抽象主题转化为具体的镜头组合逻辑。

### 4. AI Agent 驱动的系统架构设计
- 4.1 智能体作为编排中枢：Planner-Executor 架构在复杂任务拆解中的必要性。
- 4.2 Agent Runtime 与工具调用：实现检索、切分、剪辑等工具的闭环调用。
- 4.3 记忆与上下文组装：管理多源素材元数据，优化 Agent 决策质量。

### 5. 系统工程与工程管线
- 5.1 Client-Core-Server 三层架构：实现系统的高内聚与低耦合。
- 5.2 服务端能力设计：Runtime Guard、向量检索与自动化测试的工程实践。

### 6. 总结与展望
- 6.1 智能化工作流的优势：提升编排灵活性与用户意图理解。
- 6.2 当前挑战：长上下文处理效率与编排结果的艺术性评价。
